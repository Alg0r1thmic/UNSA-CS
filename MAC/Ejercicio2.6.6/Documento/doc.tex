\section{Problema}

Let z denote the vector whose entries are all ones (Using $MATLAB: z =
ones (n, 1)$ ), and let $b = H_{n} z$, where $H_{n}$ is again the $nxn$ Hilbert matrix. If
we now solve the system $H_{n}x = b$ for $x$, we should get $z$ as the solution in theory.
Using MATLAB, try solving $H_{n}x=b$ for $n = 4, 8, 12$, and $16$,
and see what you get. In each case compute the condition number $K_{2}(H_{n})$ and the
norm of the difference: $|| \hat{x}-z ||_{2} $, where $x$ is the computed solution. Calculate the
residual $\hat{r} = b - H_{n}\hat{x}, too$.

\section{Resolución}

\begin{itemize}
 \item Comando utilizado para hallar las $\hat{x}$ : \textbf{inv(hilb(n))*(hilb(n)*ones(n,1)}
 \item Comando utilizado para hallar el numbero de condición de las matrices de Hilbert: \textbf{cond(hilb(n),2)}
\end{itemize}
