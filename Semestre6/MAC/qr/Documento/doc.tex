\section{Problema y Código}

Implementar el algoritmo de QR en octave.


\begin{lstlisting}
function  D = my_qr(A,n)
	D = A;
	for i = 1 : n
		[Q,R] = qr(D);
		D = R * Q;
	end
endfunction
\end{lstlisting}
